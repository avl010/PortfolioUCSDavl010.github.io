Dice Game

The Dice Game is a virtual experience where players can roll a six-sided dice, customizing the number of rolls and enjoying the thrill of chance. Players can track their virtual monetary balance throughout the game.

How It Works

  1. Start by clicking the "Roll Dice" button.
  2. Choose the number of rolls you'd like to make.
  3. Experience the excitement as the virtual dice is rolled.
  4. Watch your virtual monetary balance change based on the dice outcome.
  5. Enjoy the game and have fun!

Process

  1. The game is structured using HTML. It has a container div with various elements inside.
  2. CSS styling is applied to the HTML elements using an external stylesheet.
  3. The JavaScript file handles the game logic, including rolling the dice and determining the outcome based on the sum of the dice.
  4. When the user interacts with the input field and clicks the "Let's Play Again!" button, the JavaScript function letsPlay( ) is triggered, updating the game outcome and final balance.
